# 📊 Customer Behavior Analysis using Python & Pandas

## 🚀 Overview
This project analyzes customer purchasing behavior using Python and Pandas. It identifies high-value customers, revenue trends, and category-wise performance.

---

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib

---

## 📂 Features
- Data cleaning and preprocessing  
- Customer classification (High / Moderate / Low value)  
- Revenue analysis by category  
- Top customer identification  
- Average order value calculation  
- Data visualization (bar chart & pie chart)  

---

## 📊 Key Insights
- Electronics category generates the highest revenue  
- High-value customers contribute most to total sales  
- Multi-category customers show higher engagement
